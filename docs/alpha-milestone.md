# Alpha Milestone — Darkstone Cavern Demo

## Scope
Playable vertical slice demonstrating all core systems.

## Content
- **1 Dungeon**: Darkstone Cavern (6 rooms, linear with branching side passage)
- **3 Enemy Types**:
  - Cave Slime (tutorial enemy — low HP, low attack)
  - Bone Archer (mid-game — balanced stats)
  - Dark Knight (boss — high HP/attack/defense)
- **Story**: Intro narration → dungeon exploration → boss dialogue → victory/defeat ending

## Win Condition
Defeat the Dark Knight and reach the exit.

## Systems Exercised
- [x] Overworld movement (room-to-room navigation)
- [x] Battle system (all 3 actions: attack/heal/flee)
- [x] Dialogue (branching, portraits, narrator)
- [x] Game state (overworld ↔ battle ↔ dialogue transitions)

## Out of Scope (Phase 2)
- Save system
- Inventory / equipment
- Multiple dungeons
- Music / SFX (placeholder only)
- Tilemap art (colored rectangles as placeholder)
