# Social Media Setup Checklist

## Account Creation Guide

This checklist must be completed by a human marketer. AI agents cannot create social media accounts.

---

## 1. Platform Accounts

### Twitter/X
- [ ] Create account @8bitrpg_game (or available variant)
- [ ] Upload profile picture (512x512 game icon)
- [ ] Upload header image (1500x500 px - game world panorama)
- [ ] Write bio: "Retro 8-bit RPG adventure in development. Classic combat, modern polish. Made with Godot 4.2. Wishlist on Steam! 🎮"
- [ ] Add link to itch.io/Steam page
- [ ] Pin welcome tweet (see WEEK1_CONTENT_CALENDAR.md)
- [ ] Create lists: Influencers, Press, Community

**Account URL:** twitter.com/8bitrpg_game (or variant)

---

### Instagram
- [ ] Create account @8bitrpg (or available variant)
- [ ] Upload profile picture (320x320 minimum)
- [ ] Write bio: "8-bit RPG | Pixel Art | Indie Game | Developer diaries and art ✨"
- [ ] Set to Professional/Creator account
- [ ] Enable insights
- [ ] Link to Linktree or main storefront

**Account URL:** instagram.com/8bitrpg (or variant)

---

### TikTok
- [ ] Create account @8bitrpg (or available variant)
- [ ] Upload profile picture (200x200 minimum)
- [ ] Write bio: "Making an 8-bit RPG one pixel at a time 🎮 Follow for dev updates! #indiegame #pixelart"
- [ ] Link to main storefront

**Account URL:** tiktok.com/@8bitrpg (or variant)

---

### YouTube
- [ ] Create channel "8-Bit RPG Official" (or similar)
- [ ] Upload channel art (2560x1440 px)
- [ ] Upload profile picture (800x800 recommended)
- [ ] Write channel description with links
- [ ] Create playlists: Dev Diaries, Gameplay, Tutorials
- [ ] Add channel trailer

**Channel URL:** youtube.com/@8bitrpg (or variant)

---

### Discord Server
- [ ] Create Discord server "8-Bit RPG Community"
- [ ] Set server icon (128x128 game icon)
- [ ] Create role hierarchy:
  - Developer
  - Moderator
  - Superfan
  - Member
- [ ] Create channel structure (see below)
- [ ] Set up welcome bot (Dyno or Carl-bot)
- [ ] Set up moderation bot
- [ ] Create #rules-faq channel content
- [ ] Create #welcome message
- [ ] Add widget to website/store pages

## 2. Account Management Setup

### Scheduling Tools
- [ ] Create Buffer account (free tier)

### Analytics
- [ ] Enable Twitter Analytics
- [ ] Enable Instagram Insights
- [ ] Enable YouTube Studio
- [ ] Set up Google Analytics for website/store

### Link Management
- [ ] Create Linktree account (free)
- [ ] Add links: Steam wishlish, Discord, Twitter,itch.io
- [ ] Use Linktree in all bio links

## 4. Required Before Launch

### Brand Assets (Priority)
Before posting, ensure you have:
- [ ] Game icon/logo (512x512, 128x128, 64x64, 32x32)
- [ ] Twitter header (1500x500)
- [ ] YouTube channel art (2560x1440)
- [ ] 3-5 gameplay screenshots
- [ ] 2-3 gameplay GIFs

If assets don't exist, coordinate with development team to:
1. Capture screenshots from playable build
2. Create logo/design assets
3. Record GIF clips from gameplay

---

## 5. Security & Access

### Account Security
- [ ] Use a shared password manager (1Password, Bitwarden)
- [ ] Enable 2FA on all accounts
- [ ] Document login credentials securely
- [ ] Set up backup codes

### Team Access
- [ ] Add team members as account managers where applicable
- [ ] Document who has access to which platforms
- [ ] Create contact list for emergencies

---

## 6. Launch Day Checklist

Before first post:
- [ ] All accounts created and verified
- [ ] Profile pictures uploaded
- [ ] Bios written with links
- [ ] Welcome posts drafted in scheduling tool
- [ ] Discord channel structure complete
- [ ] Team briefed on launch plan
