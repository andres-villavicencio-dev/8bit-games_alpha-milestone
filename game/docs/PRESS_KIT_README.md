# 8-Bit RPG - Press Kit README

## Quick Start

Welcome to the 8-Bit RPG press kit. This folder contains all assets and information needed for media coverage, influencer content creation, and promotional materials.

### Folder Structure

```
press-kit/
├── README.txt               # This quick-start guide
├── fact-sheet.pdf          # One-page game overview (print-friendly)
├── logo/                   # Logo assets for all use cases
├── screenshots/            # Gameplay screenshots (960x540)
├── screenshots-hd/         # High-res screenshots (1920x1080)
├── gifs/                   # Animated gameplay clips
├── videos/                 # Trailers and video content
├── artwork/                # Character art, environments, concept art
└── audio/                  # Sample soundtrack and sound effects
```

### Recommended Use

**For Articles/Reviews:**
- Use `fact-sheet.pdf` for quick reference
- Use `screenshots-hd/` for high-quality images
- Use `logo/logo-primary.png` for headers

**For Social Media:**
- Use `gifs/` for engaging content
- Use `screenshots/` for tweets/posts
- Use `logo/logo-icon-512.png` for avatars

**For Video Content:**
- Use `videos/trailer-announcement.mp4` for footage
- Use `audio/` for background music
- Use `artwork/` for thumbnails

### Logo Usage Guidelines

✅ **DO:**
- Use primary logo on light backgrounds
- Use dark version on dark backgrounds
- Maintain aspect ratio when scaling
- Use icon for small avatars/favicons

❌ **DON'T:**
- Stretch or distort logo
- Change colors
- Add effects or filters
- Use low-resolution versions
