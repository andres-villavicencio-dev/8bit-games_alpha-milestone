# 8-bit RPG — Founding Engineer's Monorepo

Built with **Godot 4** — MIT licensed, pixel-perfect, no royalties.

## Structure

- `game/` — Godot 4 project (main game)
- `tools/` — Asset pipeline scripts
- `docs/` — Game design documents
- `tests/` — Quality assurance resources and test suite

## Setup

1. Install [Godot 4](https://godotengine.org/download/) (v4.x)
2. Open `game/project.godot` in Godot editor
3. Press F5 to run

## Architecture

Core systems: Overworld → Battle → Dialogue
